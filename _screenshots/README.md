# Skrinshotlar uchun vaqtinchalik papka

Bu papka faqat **rasmlarni kesish** uchun vaqtinchalik ishlatiladi.
Kesib bo'lingach, skrinshotlar bu yerdan o'chiriladi.

## Foydalanish

1. Bu papkaga skrinshotlarni yuklang (drag-and-drop)
2. Fayl nomlari aniq bo'lsin. Misollar:
   - `2.1-2.9.png` — Imtiyoz belgilari (9 ta)
   - `3.1-3.10.png`, `3.11-3.20.png` ... — Ta'qiqlovchi belgilari
3. Kiroga qaysi guruh ekanini ayting
4. Kiro avtomatik kesib, `images/<guruh>/` ga joylaydi va papkani tozalaydi
